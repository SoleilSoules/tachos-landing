# Tachos — HANDOFF (для нового чата)

Боевая вёрстка лендинга **tachos.ru** по Figma-концепту. Дата передачи: 2026-06-13.

## Запуск
```
cd ~/tachos-ru && npm run dev   # → localhost:3000
```
Открывать Гоше в Safari: `open -a Safari http://localhost:3000/`.
Стек: **Next.js 16 + React 19 + TS strict + Tailwind 3.4** (как у Вадима/monte_hub).
Токены — CSS-переменные в `app/globals.css` → `tailwind.config.ts` (bg/ink/surface/surface2/accent[.bright/.hot/.warm]/inverted). Шрифт **Onest**. Контент в `lib/content.ts`.

## ЧТО СДЕЛАНО (верх лендинга, ~1-в-1)
- **Nav** (`components/sections/Nav.tsx`) — full-width (лого слева/CTA справа у краёв экрана), пункты Кейсы/Отзывы/Контакты/О нас/Медиа.
- **Hero** (`Hero.tsx`+`HeroPrompt.tsx`, фон в `app/page.tsx`) — iPhone-мокап тёплый цельный (виньетка вертикальная, не радиальная), H1, оранжевый подзаголовок, инпут (pill, мигающая оранжевая каретка, **waveform** в белой кнопке — НЕ микрофон!), чипы светло-серые (surface2), entrance-reveal.
- **LogoWall** — 7 лого, крайние уходят за край экрана (-mx).
- **Founder** — карточка, «Написать Вадиму» белая, «Презентация Вадима» 1 строка с play в кружке, фото справа, оранжевое свечение, magnetic-кнопка.
- **Cases** (`Cases/CaseCard/CaseTabs/CasesHeadingClient`) — заголовок с одометром «40», табы (Избранные/Для банков/eCommerce/Для маркетплейсов/Геймдев), **4 кейса** 2×2 без блюра, кнопка «Все кейсы».
- **Reviews** (`Reviews.tsx`) — «Отзывы клиентов», мозаика: аудио-карточка с плеером+waveform, текстовые, видео (оранжевая), портретная (Дарья, круг с ring). Копирайт по нашим проектам (Хайс/Складно/Maginary).
- **Форма-конструктор** (`components/compose/`) — письмо-оверлей: умный typewriter (не затирает ручные правки), валидация контакта+согласия, статусы отправки (sending/success/error), focus-trap, draft в localStorage, плавающая кнопка снизу при скролле, **курсор-компаньон** (плектр с глазами). Открытие из любого `data-compose` + hero-инпута. Deep-link `?compose[=type]`.
- Анимации: reveal секций, одометр, float мокапа, magnetic CTA, hover карточек. Все уважают `prefers-reduced-motion`.
